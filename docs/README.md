# Docs Index

This folder contains non-runtime artifacts.

## Structure
- `docs/decisions` - architecture and business decision records
- `docs/planning` - roadmaps, checklists, and implementation plans
- `docs/reviews` - external reviews and audits
- `docs/research` - brainstorming transcripts and prompts
- `docs/archive` - deprecated prototypes and historical code

## Note
Runtime code lives in `apps/api`.
